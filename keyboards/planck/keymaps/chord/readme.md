# Chord layout

Building on top of the default Planck layout, a few other chord layers are added.

Inspired by [this](https://gist.github.com/pocketkk/ab50ff5012b8bf779d6267047e0e6bbc#file-keymap-c-L49) keymap.

## Concept
The concept is to have the left area of keyboard to activate layers, and the right area to select a key.
The left area refers to the first 4 alpha keys of each row, with the right area mostly emulating a numpad-like layout.

## How to use

The default Planck layers are activated by using the `LOWER` and `UPPER` keys (as usual),
whereas the chord layers are activated by holding the respective hotkeys (listed below)
